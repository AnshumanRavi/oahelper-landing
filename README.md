# 🚀 OAHelper - Premium Landing Page

> **Practice the exact online assessments that land your first job.**

OAHelper is a high-end, sleek, and dynamic landing page built to showcase an online assessment preparation platform. Designed with a dark "Cyberpunk/Premium SaaS" aesthetic, it utilizes glassmorphism, fluid animations, and a rich color palette to immediately capture user attention and establish trust.

## ✨ Features

- **Hero Section with Live Mock IDE:** A fully floating, 3D-styled code editor mockup with passing test cases and dynamic floating achievement bubbles.
- **Floating AI Mascot:** A fun, interactive CSS-based robotic companion that waves, displays tips, and provides an engaging micro-interaction for visitors.
- **Glassmorphism & Glow Effects:** Premium, frosted-glass UI components with intricate backdrop-blurs and subtle, animated gradient borders.
- **Interactive Pricing Toggle:** Seamlessly switch between Monthly and Annual pricing with dynamic, glowing price card updates.
- **Infinite Marquee:** A smooth, auto-scrolling company logo section featuring top-tier tech companies.
- **Framer Motion Animations:** Buttery-smooth entrance, scroll, and hover animations applied to nearly every component.

## 🛠 Tech Stack

- **React 18**
- **Vite**
- **Framer Motion** (for advanced animations and scroll reveals)
- **Vanilla CSS** (Custom properties, grid, flexbox, keyframes)
- **React Icons**

## 🚀 Getting Started

1. **Clone the repository**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

This project avoids generic UI frameworks in favor of hand-crafted CSS. Every shadow, blur, and gradient is precisely tuned to create a heavy "tech-focused" and premium aesthetic, ensuring that the landing page looks as high-quality as the service it provides.
